# art-AIedu-asaas

少儿美术机构 AI 招生增长系统。

## 一句话定位

帮助少儿美术机构每天生成招生内容，并通过儿童画作 AI 测评把家长转化为试听预约。

## 核心闭环

1. 老师上传课堂作品和课程主题
2. AI 生成小红书、朋友圈、短视频脚本
3. 内容引导家长进入免费画作测评
4. 家长上传孩子作品并留下联系方式
5. AI 生成测评报告和课程建议
6. 系统引导家长预约试听
7. 老师获得线索、测评摘要和跟进话术
8. 校长查看内容、测评、预约、成交数据

## 第一版模块

- 内容生成工作台
- 儿童画作测评页
- 试听预约页
- 老师线索后台
- 校长数据看板

## 目标客户

优先服务少儿美术单校区和小型连锁机构，尤其是招生竞争激烈、老师缺内容能力、校长重视私域转化的机构。

## 早期收费建议

试点阶段建议采用“AI 工具 + 运营陪跑”模式：

- 试点版：999-1999 元/月
- 标准版：1999-3999 元/月
- 后续 SaaS 版：按校区、账号数、测评次数和数据能力分层

## 当前最重要验证

先验证 5 家机构是否愿意持续付费使用：

- 每天内容生成
- 画作测评获客
- 试听预约转化
- 老师跟进话术
