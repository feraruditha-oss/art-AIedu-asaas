# 多入口获客系统设计

## 目标

把 art-AIedu-asaas 从“少儿画作测评工具”升级为“艺术机构全客群增长系统”。

系统不只服务少儿招生，也帮助机构开发成人美术、银发兴趣班、异业合作、企业年会艺术体验等收入入口，让机构老板和独立美术老师长期依赖。

## 核心定位

为艺术机构持续生成可投放、可转化、可复用的获客活动。

每个获客活动都应自动生成：

- 活动落地页
- 小红书文案
- 朋友圈文案
- 抖音/视频号脚本
- 海报标题
- 表单字段
- 测评/规划报告
- 预约入口
- 老师跟进话术
- 数据看板

## 一、少儿招生入口

### 1. 绘画潜能倾向测评

承接家长“孩子有没有天赋”的需求。

注意不要直接承诺判断天赋，建议表达为：

- 绘画潜能倾向
- 表达欲
- 观察力
- 想象力
- 色彩敏感度
- 画面组织能力

转化路径：

作品上传 -> AI 报告 -> 推荐课程 -> 预约试听。

### 2. 色彩感知测评

承接家长“孩子色感好不好”的需求。

报告维度：

- 色彩敏感度
- 色彩对比
- 情绪表达
- 配色习惯
- 材料使用

适合引流平台：

- 小红书
- 朋友圈
- 视频号

### 3. 创作力挑战

用主题活动吸引家长提交作品。

主题例子：

- 如果动物开了一家店
- 未来城市
- 我的梦境花园
- 机器人朋友
- 节日里的家

报告维度：

- 联想力
- 故事性
- 角色关系
- 画面完整度
- 主动表达

### 4. 专注力观察测评

承接家长“想提升专注度”的需求。

注意合规表达：

- 不说保证提升专注力
- 不说治疗注意力问题
- 使用“课堂观察”“专注表现”“学习习惯建议”

输入字段：

- 绘画持续时间
- 是否能完成作品
- 是否愿意听老师引导
- 是否频繁换主题
- 家长观察描述

### 5. 审美表达测评

承接家长对审美、表达、自信的需求。

适合用于高客单价课程转化。

## 二、成人美术入口

### 1. 成人零基础学习规划

目标人群：

- 白领
- 宝妈
- 大学生
- 自由职业者
- 想发展兴趣副业的人

常见需求：

- 放松减压
- 培养审美
- 学水彩/油画/插画
- 做手账/作品集
- 发展副业

系统入口：

兴趣问卷 -> 学习目标分析 -> 推荐课程路径 -> 预约体验课。

### 2. 成人艺术沙龙

适合非长期课包收入。

活动形式：

- 周末水彩沙龙
- 油画体验夜
- 香薰 + 绘画
- 咖啡馆写生
- 花艺 + 绘画
- 情绪疗愈绘画

系统应生成：

- 活动页
- 报名表
- 海报文案
- 小红书种草文案
- 朋友圈招募话术
- 成本和报价建议

## 三、银发美术入口

### 1. 退休生活艺术兴趣课

目标人群：

- 退休人群
- 社区老年学员
- 子女为父母报名
- 老年大学周边人群

常见需求：

- 丰富退休生活
- 线下社交
- 手眼协调
- 作品成就感
- 家属认可

课程方向：

- 国画花鸟
- 水彩花卉
- 书画入门
- 轻松速写
- 节气主题创作

系统应生成：

- 社区招募文案
- 子女转发文案
- 体验课活动页
- 慢节奏课程介绍
- 老师跟进话术

## 四、异业合作入口

### 1. 本地异业合作

合作对象：

- 书店
- 咖啡馆
- 商场
- 社区
- 儿童摄影
- 绘本馆
- 托管机构
- 舞蹈/口才/运动机构
- 花店
- 香薰/手作店

合作形式：

- 联合体验课
- 周末艺术快闪
- 亲子艺术工坊
- 会员互推
- 消费满额赠艺术体验
- 作品展陈列

系统应生成：

- 合作提案
- 合作权益说明
- 联合海报文案
- 活动流程
- 报名表
- 分成记录
- 合作效果数据

## 五、企业活动入口

### 1. 公司年会艺术体验

目标客户：

- 企业 HR
- 行政负责人
- 团建公司
- 商会/协会
- 社群组织者

活动形式：

- 企业年会共创画
- 团队油画体验
- 女神节艺术沙龙
- 员工亲子日
- 客户答谢艺术活动
- 企业文化墙共创

系统应生成：

- 企业活动方案
- 活动报价单
- 执行流程
- 物料清单
- 销售跟进话术
- 成交合同要点

## 六、月度获客日历

机构长期使用的关键，是让系统每月自动安排可执行活动。

示例：

- 第 1 周：绘画潜能测评
- 第 2 周：色彩感知挑战
- 第 3 周：成人水彩沙龙
- 第 4 周：亲子艺术工坊
- 节日前：节日主题创作营
- 月底：作品成长档案 + 老带新邀请
- 淡季：企业团建和异业合作开发

## 七、数据看板新增指标

除了少儿试听数据，还应加入：

- 活动创建数量
- 不同入口线索数
- 少儿线索数
- 成人线索数
- 银发线索数
- 异业合作线索数
- 企业活动询盘数
- 活动成交金额
- 合作方带来线索
- 老带新转介绍数

## 八、产品壁垒

新增多入口后，系统壁垒不再只是 AI 测评，而是：

- 少儿美术测评模型
- 成人学习规划模型
- 银发艺术课程模板
- 本地异业合作提案库
- 企业艺术活动方案库
- 内容生成模板库
- 老师/校长跟进话术库
- 活动转化数据

这会让机构更难替换，因为它绑定的是机构每月招生、活动收入和本地合作收入。
