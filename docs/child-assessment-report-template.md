# 少儿艺术测评报告模板

## 使用目标

让少儿艺术测评从“简单点评”升级为“可转化、可教学衔接、可同步老师端”的专业报告。

## 报告标题

少儿艺术发展阶段观察报告

副标题：

基于孩子年龄、作品图像、学画经历和家长关注点生成。

## 一、作品概览

示例：

这幅作品呈现出较完整的场景意识。孩子不仅画出了单个形象，还尝试把角色、环境、活动和装饰细节放进同一个画面里，说明孩子已经开始从“画一个物体”进入“组织一个故事场景”的阶段。

## 二、年龄阶段分析

### 3-4 岁

重点观察：

- 是否愿意涂画
- 是否喜欢尝试材料
- 是否能通过线条和色块表达情绪
- 是否出现简单形象意识

家长解释：

这个阶段不建议过早要求画得像，更重要的是保护孩子的表达欲和材料探索兴趣。

### 5-6 岁

重点观察：

- 是否出现符号化人物、房子、动物等形象
- 是否能围绕一个主题画画
- 是否愿意用颜色表达情绪
- 是否开始讲述画里的故事

家长解释：

这个阶段的画面可能看起来不够有秩序，但常常包含孩子自己的想法和叙事。

### 7-8 岁

重点观察：

- 是否能观察对象特征
- 是否能安排角色关系
- 是否能完成较完整主题
- 是否开始有构图意识

家长解释：

这个阶段可以逐步引导孩子从自由表达走向观察、组织和主题表达。

### 9-12 岁

重点观察：

- 造型是否稳定
- 空间层次是否清楚
- 画面主次是否明确
- 主题是否完整
- 是否能持续处理细节

家长解释：

这个阶段适合进入更系统的主题创作、造型基础和材料表现训练。

## 三、五维度拆解

### 1. 色彩

分析字段：

- 色彩使用是否大胆
- 色彩对比是否明确
- 是否有情绪表达
- 主体和背景是否能区分
- 是否适合进入色彩专项训练

输出示例：

孩子在色彩上有较强主动性，能用明亮颜色强化画面重点。下一步可以学习冷暖、明暗和主次色关系，让画面更稳定。

### 2. 线条

分析字段：

- 线条是否大胆
- 是否有控制力
- 是否有节奏变化
- 是否能服务造型和结构
- 是否能表达细节

输出示例：

孩子愿意使用线条处理细节，说明观察和表达意愿较强。后续可以引导线条从装饰性重复，逐步走向结构和动态表达。

### 3. 造型

分析字段：

- 是否能形成明确形象
- 是否抓住对象特征
- 人物、动物、建筑是否有区分
- 是否有比例意识
- 是否有角色个性

输出示例：

孩子已经能通过服饰、动作和表情区分角色。下一步可以继续训练人物动态、比例关系和角色之间的互动。

### 4. 构图

分析字段：

- 主体是否清楚
- 画面是否完整
- 前景、中景、背景是否出现
- 视觉中心是否明确
- 主次关系是否清楚

输出示例：

孩子已经开始安排画面空间，而不是把元素简单平铺。下一步可以加强视觉中心和主次取舍，让故事重点更清晰。

### 5. 画面组织

分析字段：

- 是否有主题
- 是否有故事线
- 是否有角色关系
- 是否能整合多个元素
- 是否能取舍
- 是否有画面节奏

输出示例：

作品呈现出较强的主题组织能力，孩子能把环境、角色和活动整合为完整场景。后续适合进入主题创作课程，训练故事线、空间层次和画面节奏。

## 四、学习建议

建议输出 3 条：

1. 保留孩子当前最明显的表达优势
2. 给出一个具体训练方向
3. 给出适合课程或活动

示例：

- 继续保护孩子主动讲故事的表达欲。
- 课堂中增加构图草稿和主次取舍训练。
- 建议体验高阶主题创作课，由老师现场观察孩子构思、起稿和细节推进过程。

## 五、推荐课程

课程推荐不能只按年龄，还要结合测评维度。

示例：

- 色彩敏感明显：创意色彩课
- 表达欲强：主题创作课
- 造型基础弱：观察写生课
- 细节耐心好：综合材料课
- 空间意识强：高阶主题创作课

## 六、体验课老师同步内容

报告生成后同步到老师后台：

- 作品图片
- 年龄阶段判断
- 五维度分析
- 五维能力图：色彩、线条、造型、构图、画面组织
- 家长关注点
- 推荐课程
- 试听观察任务

## 七、试听观察任务

老师试听时重点观察：

- 孩子是否能先构思再下笔
- 是否能听懂老师的构图建议
- 是否能持续完成作品
- 是否愿意表达自己的画面故事
- 遇到困难时是否愿意尝试调整

## 八、合规表达

避免：

- 你的孩子很有天赋
- 孩子没有绘画天赋
- 报课后一定能提升专注力
- 一定能获奖或考级

建议：

- 孩子呈现出某方面的潜能倾向
- 当前作品表现出较强表达欲
- 后续可通过课程持续观察和引导
- 体验课老师会结合现场表现补充判断
