# 开发任务拆分

## Phase 0：准备

- 确定品牌名、产品定位和目标机构画像
- 整理 300-500 张真实儿童作品样本
- 邀请 3-5 位美术老师共创测评标准
- 选择 5 家机构做试点

## Phase 1：产品原型

- 设计少儿艺术测评页
- 设计测评报告页
- 设计试听预约页
- 设计老师线索后台
- 设计内容生成工作台
- 设计校长数据看板

## Phase 2：基础系统

- 搭建账号登录
- 搭建机构和校区管理
- 搭建作品上传能力
- 搭建测评记录数据库
- 搭建线索管理
- 搭建试听预约管理

## Phase 3：AI 能力

- 接入图像理解模型
- 编写儿童画作测评提示词
- 编写课程推荐规则
- 编写内容生成提示词
- 编写老师跟进话术提示词
- 支持老师二次编辑 AI 输出

## Phase 4：数据看板

- 内容生成统计
- 测评提交统计
- 预约试听统计
- 来源平台统计
- 线索状态统计
- 简单转化漏斗

## Phase 5：试点迭代

- 让 5 家机构真实使用 2-4 周
- 每周收集老师反馈
- 每周收集校长反馈
- 优化测评报告话术
- 优化内容模板
- 优化预约转化路径
