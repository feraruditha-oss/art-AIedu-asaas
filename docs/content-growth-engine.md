# 内容增长引擎设计

## 目标

把内容生成从“单次写文案”升级为“长期可滚动使用的招生内容系统”。

机构和独立老师需要的不是几条通用文案，而是：

- 每天有内容可发
- 每个平台内容不一样
- 每家机构内容不重复
- 能结合真实作品和课堂素材
- 能不断生成新角度
- 能通过数据反馈优化选题
- 能最终带来测评、试听和成交线索

进一步升级为动态内容库：

- 每日采样平台热点和同城热词
- 每周复盘机构内容表现
- 节日前 30 天自动启动内容预热
- 根据历史相似度和转化数据自动换角度
- 为不同机构生成差异化内容，避免同质化

## 一、内容库结构

内容库不能只存成品文案，要拆成可组合的模块。

### 1. 选题库

按主题分类：

- 家长误区
- 年龄段科普
- 作品点评
- 色彩感知
- 创作力
- 专注力观察
- 课堂过程
- 学员成长
- 节日活动
- 亲子体验
- 成人艺术沙龙
- 银发兴趣课
- 异业合作
- 企业团建

### 2. 钩子库

用于视频开头、封面标题、小红书首句。

例：

- 孩子画得乱，不一定是不会画
- 别急着纠正孩子画不像
- 一张画能看出孩子的表达欲
- 这个年龄段最该保护的不是技巧
- 孩子有没有创作力，看这几个细节

### 3. 作品点评库

按五维度拆：

- 色彩
- 线条
- 造型
- 构图
- 画面组织

每个维度要有大量表达变体。

### 4. 年龄段解释库

按年龄段：

- 3-4 岁
- 5-6 岁
- 7-8 岁
- 9-12 岁
- 成人零基础
- 银发兴趣学习

### 5. 平台结构库

每个平台有不同模板：

- 抖音：3 秒钩子 + 分镜 + 口播 + 字幕 + 同城引导
- 小红书：搜索标题 + 封面大字 + 正文 + 作品拆解 + 私信引导
- 视频号：真实课堂 + 老师点评 + 家长转发文案 + 小程序入口
- 朋友圈：课堂记录 + 作品图文 + 老师观察 + 轻转化

### 6. 转化话术库

每条内容都必须有转化出口：

- 免费少儿艺术测评
- 色彩感知测评
- 创作力挑战
- 专注力观察
- 试听预约
- 亲子工坊报名
- 成人艺术沙龙报名
- 企业团建咨询

注意：平台内容不要默认生成“评论区留言测评”“私信领取”“加微信”等硬诱导表达。更稳妥的方式是设计真实讨论型互动，再通过主页入口、组件、小程序、企业号工具或老师承接转化。

更推荐：

- 你家孩子画画时，是更喜欢画细节，还是先讲故事？
- 你观察到孩子画里最常出现什么？
- 如果孩子到了 7-8 岁还只画单个物体，可以重点看画面组织。
- 附近家长可以带作品到校区，让老师现场看一看阶段。

不推荐：

- 评论区留言“测评”
- 私信我领资料
- 加微信免费测
- 转发到群领取

### 7. 本地话术库

美术教培是强本地行业，核心服务半径通常在校区周边 3-5 公里。内容库必须内置本地引导话术，而不是生成全国泛流量文案。

本地字段：

- 城市
- 区域
- 商圈
- 校区名称
- 周边社区
- 周边学校/幼儿园
- 交通地标
- 招生半径

本地引导示例：

- 滨江区附近的家长，可以带孩子作品来校区做一次少儿艺术测评。
- 校区周边 3-5 公里的家庭，周末可以预约一节主题体验课。
- 住在 XX 社区附近的家长，如果孩子喜欢画画，可以先上传作品看看阶段。
- XX 商圈周末亲子活动，适合 5-8 岁孩子体验创意美术。

## 二、原创和去重机制

平台越来越重视原创，系统必须避免给不同机构生成高度相似内容。

### 1. 机构专属内容指纹

每家机构建立专属字段：

- 城市
- 校区风格
- 老师表达习惯
- 课程体系
- 学生作品风格
- 常用活动
- 品牌语气
- 目标家长画像

生成内容必须结合这些字段，减少同质化。

### 2. 内容相似度检查

系统应记录每次生成结果：

- 标题
- 正文
- 视频脚本
- 封面文案
- 发布时间
- 使用机构
- 使用平台

生成新内容前检查：

- 标题相似度
- 首句相似度
- 段落结构相似度
- 关键词重复度
- 选题重复频率
- 本地关键词使用频率

相似度过高时自动重写。

### 3. 多层改写

同一主题至少从 5 个角度改写：

- 家长痛点角度
- 老师专业角度
- 孩子成长角度
- 作品细节角度
- 课程价值角度

### 4. 素材绑定

优先让内容基于真实素材生成：

- 真实学生作品
- 真实课堂照片
- 老师真实点评
- 家长真实问题
- 当地活动场景

真实素材越多，内容越不容易重复。

## 三、算法推荐机制理解

系统不应该承诺“保证爆款”，但要按平台推荐逻辑提高命中率。

### 抖音

核心影响：

- 前 3 秒停留
- 完播率
- 互动率
- 同城兴趣匹配
- 视频画面冲击
- 评论区讨论

内容生成重点：

- 强开头
- 短句口播
- 画面变化
- 明确冲突
- 同城标签
- 体验课引导
- 校区周边 3-5 公里引导

### 小红书

核心影响：

- 搜索匹配
- 标题关键词
- 收藏率
- 评论互动
- 内容真实感
- 图片封面清晰度

内容生成重点：

- 搜索型标题
- 家长痛点关键词
- 封面大字
- 真实作品案例
- 结构化正文
- 私信关键词引导
- 区域关键词和附近社区词

### 视频号

核心影响：

- 熟人关系传播
- 完播
- 点赞/转发
- 微信私域承接
- 公众号/小程序连接

内容生成重点：

- 真实课堂
- 老师温和表达
- 家长愿意转发
- 小程序测评入口
- 企微跟进
- 社区、校区、熟人转发话术

## 四、爆款文案生成框架

### 1. 标题公式

```text
家长痛点 + 反常识判断
```

例：

孩子画得乱，不一定是不会画

更强示例：

- 孩子画得很满，不一定是乱，可能是在努力讲故事
- 别急着纠正比例，5-6 岁最该保护的是表达欲
- 孩子总画同一个东西，家长真正该担心的是这个
- 画得不像不可怕，可怕的是孩子不敢表达
- 一张画里东西很多，老师先看这 3 个点

```text
年龄段 + 专业提醒
```

例：

5-6 岁学美术，最不该急着纠正比例

```text
作品细节 + 能力判断
```

例：

这张画里的小细节，能看出孩子的创作力

### 2. 正文公式

```text
家长常见疑问
-> 真实作品
-> 老师专业拆解
-> 年龄段解释
-> 下一步建议
-> 测评/试听入口
```

### 3. 短视频公式

```text
3 秒钩子
-> 展示作品/课堂过程
-> 老师一句判断
-> 解释孩子能力
-> 邀请测评/试听
```

## 五、系统功能建议

### 1. 内容日历

每家机构每月自动生成：

- 8 条小红书
- 8 条朋友圈
- 8 条短视频脚本
- 4 个活动招募内容
- 4 条老带新内容

### 2. 内容去重评分

生成后展示：

- 原创度
- 平台适配度
- 家长痛点强度
- 转化强度
- 与历史内容相似度

### 3. 内容资产沉淀

每家机构形成自己的：

- 作品库
- 标题库
- 话术库
- 高转化内容库
- 本地活动库

### 4. 数据反馈

老师发布后回填：

- 曝光
- 点赞
- 收藏
- 评论
- 私信
- 测评提交
- 试听预约
- 成交

系统根据数据优化下一批内容。

## 六、第一版先做什么

第一版内容增长引擎不需要复杂算法，先做这 5 件事：

1. 建立 100 个少儿美术选题
2. 建立 300 个标题钩子
3. 建立 50 个作品点评模板
4. 每次生成记录历史内容
5. 生成前做相似度提醒

等有 20-50 家机构使用后，再用真实数据优化模板和推荐策略。
