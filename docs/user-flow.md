# 用户流程

## 机构老师流程

1. 登录后台
2. 上传今日课堂作品或输入课程主题
3. 选择发布平台：小红书、朋友圈、抖音、视频号
4. AI 生成内容
5. 老师修改后发布
6. 家长通过内容进入测评页
7. 老师在后台接收线索
8. 根据 AI 话术跟进试听

## 家长流程

1. 从内容平台看到机构内容
2. 点击或扫码进入免费画作测评
3. 上传孩子画作
4. 填写年龄、学画经历、关注点和联系方式
5. 查看 AI 测评报告
6. 查看适合课程
7. 预约试听
8. 接收老师跟进

## 校长流程

1. 查看各老师内容生成情况
2. 查看各平台测评线索
3. 查看试听预约数据
4. 查看成交转化数据
5. 判断哪些内容、课程和老师带来更好转化

## 核心转化链路

内容曝光 -> 免费测评 -> 课程建议 -> 试听预约 -> 老师跟进 -> 到店成交
