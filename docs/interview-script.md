# 试点机构访谈提纲

## 访谈目标

判断少儿美术机构是否真的需要“内容生成 + 画作测评 + 试听预约 + 老师跟进”这套系统，并验证他们是否愿意付费试用。

## 适合访谈对象

- 美术机构校长
- 校区负责人
- 招生负责人
- 课程顾问
- 一线美术老师

优先访谈校长和招生负责人，因为他们最关心招生结果和付费决策。

## 访谈时长

建议 30-45 分钟。

## 开场话术

我们正在做一个少儿美术机构 AI 招生增长系统，主要帮助机构解决三件事：老师每天不知道发什么内容、家长咨询后不容易转化、试听线索跟进不够系统。

今天不是卖产品，主要想了解你们真实招生流程和痛点。如果后面产品适合，也可以邀请你们成为第一批试点机构。

## 一、机构基本情况

1. 你们目前有几个校区？
2. 主要做哪个年龄段？3-6 岁、7-9 岁、10-12 岁，还是全年龄段？
3. 目前主推课程有哪些？
4. 客单价大概是多少？按月、季度、半年还是全年收费？
5. 每月大概新增多少试听？成交多少？

## 二、招生现状

1. 现在主要靠哪些渠道招生？
2. 小红书、抖音、视频号、朋友圈分别有没有做？
3. 哪个渠道带来的家长质量最好？
4. 现在获客最难的地方是什么？
5. 家长从看到内容到预约试听，中间通常会卡在哪里？

## 三、内容生产痛点

1. 老师现在每周会发几条内容？
2. 内容是谁写？老师、校长、运营，还是外包？
3. 老师最不会写哪类内容？
4. 课堂作品和过程照片有没有稳定沉淀？
5. 你觉得什么内容最容易引来家长咨询？

## 四、家长咨询和转化

1. 家长最常问的 5 个问题是什么？
2. 家长最常见的犹豫点是什么？
3. 价格、时间、距离、孩子兴趣，哪个影响最大？
4. 你们现在怎么邀约试听？
5. 试听后不报名，通常是什么原因？

## 五、画作测评需求

1. 如果家长上传孩子作品，你觉得他们会不会愿意看一份测评报告？
2. 家长最想从测评报告里看到什么？
3. 你觉得报告应该偏专业，还是偏容易理解？
4. 哪些评价不能说？哪些话家长会反感？
5. 测评报告后推荐试听，你觉得自然吗？

## 六、老师后台需求

1. 老师拿到一个家长线索后，最需要看到什么信息？
2. AI 自动给老师生成跟进话术，你觉得有用吗？
3. 老师是否愿意在后台更新跟进状态？
4. 你希望校长能看到哪些数据？
5. 哪些功能对老师来说太复杂，可能不愿意用？

## 七、付费意愿

1. 如果系统能帮你们每天生成内容、获得测评线索、预约试听，你觉得有价值吗？
2. 你愿意先用 2-4 周试点吗？
3. 如果试点有效，每月 999-1999 元能接受吗？
4. 如果加上运营陪跑和数据复盘，每月 2999 元以上能接受吗？
5. 你希望按月付、按年付，还是按效果付？

## 八、试点邀请话术

我们第一阶段只找 5 家机构试点，不做大规模开放。试点期间我们会帮你们搭建测评入口、内容模板和老师跟进流程，也会每周一起复盘数据。

如果你愿意参与，我们需要你们提供：

- 真实课堂作品样本
- 课程介绍
- 常见家长问题
- 老师点评案例
- 每周真实使用反馈

我们会帮你们验证：内容是否更容易产出，测评是否能带来线索，线索是否能转成试听。

## 判断标准

### 高优先级试点机构

- 每月有稳定招生需求
- 愿意提供真实作品和反馈
- 老师有内容生产压力
- 校长愿意看数据
- 能接受 999 元/月以上试点价格

### 暂缓试点机构

- 完全不做线上内容
- 没有专人跟进线索
- 不愿意提供样本
- 只想免费试用
- 对 AI 过度期待但不愿参与反馈
